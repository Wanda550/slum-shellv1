# C++ Unix Shell

**slum (Simple Linux Unix Machine)** is a minimal Unix-like **shell** written in **C++** to explore Linux **process management**, **system programming**, and **command execution**.

### Features

- Command execution using fork/exec
- Built-in commands (cd, exit, pwd)
- Input parsing and tokenization
- Error handling
- Modular design

### Why this project?

This project was built to explore my understanding of:

- Linux system calls
- Process lifecycle
- File descriptors
- C++ memory management
- Shell design principles
